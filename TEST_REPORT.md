## 🔍 发现的问题

### 问题1: 评论提交后页面跳转
**描述**: 点击发送按钮后，页面跳转到了"我的"页面，而不是停留在空间页面。

**可能原因**:
1. 发送按钮的点击事件冒泡到了导航栏
2. 事件处理逻辑需要优化

**建议修复**:
```tsx
// 在发送按钮的点击事件中添加阻止冒泡
<button
  onClick={(e) => {
    e.stopPropagation();
    submitComment();
  }}
>
```
