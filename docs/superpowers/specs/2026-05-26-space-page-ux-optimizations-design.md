# 动态页（Space）UI 优化设计

**目标**：优化「动态」页面体验，满足以下需求：

1. 动态内容图片可点击查看大图（支持多图切换、可关闭）。
2. 内容发布时间显示到分钟（例如：5月25日14:30）。
3. 右上角发布入口图标简化为相机 icon（icon-only）。
4. 去掉顶部左侧用户头像与昵称，统一为“动态”标题。

## 范围

- 仅涉及前端页面 `src/pages/Space/index.tsx` 的 UI 与交互，不改 Supabase 表结构。
- 图片预览采用轻量 Modal 组件，优先满足“可看大图 + 可关闭 + 可切换”，不做复杂手势缩放。

## 交互设计

### 图片大图预览

- 入口：点击动态九宫格任意图片。
- 展示：全屏遮罩（黑底），居中展示图片（contain）。
- 关闭：右上角关闭按钮（X）；点击遮罩空白处关闭；支持 Esc。
- 切换：左右按钮切换上一张/下一张（在 2+ 图片时显示）。

### 时间格式

- `created_at` 显示为 `M月D日HH:mm`，本地时区。
- 适用范围：动态列表、评论列表。

### 顶部栏

- 左侧：固定标题“动态” + 刷新按钮。
- 右侧：发布按钮改为相机 icon-only（`Camera`），保持原发布逻辑不变。

## 组件与文件

- 新增：`src/components/ImagePreviewModal/index.tsx`
  - 职责：承载图片大图预览、切换、关闭与键盘事件。
- 修改：`src/pages/Space/index.tsx`
  - 接入 `ImagePreviewModal`
  - 调整 header 布局
  - 更新时间格式函数

## 验收标准

- 点击动态图片可打开大图预览，显示清晰、不变形，可关闭。
- 发布按钮为 icon-only，相机样式简洁；点击仍可正常打开发布弹窗。
- 顶部不再展示用户头像与昵称。
- 动态与评论时间展示到分钟。
