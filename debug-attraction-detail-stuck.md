# [OPEN] attraction-detail-stuck

## 症状

- 线上环境：从首页点击景区卡片进入 `/attraction/:id` 后一直转圈，详情页无法加载出来。

## 假设（可证伪）

1. **详情接口请求失败但未正确落入错误态**：请求报错/返回空，但 `loading` 未被置为 false，导致永远转圈。
2. **详情接口参数不匹配**：路由 `id` 与数据库 `attractions.id` 类型/格式不一致（例如数字 vs uuid/字符串），导致 `.single()` 报错或无数据。
3. **线上环境变量/域名问题**：详情页请求走了错误的 Supabase URL（例如带了 `/rest/v1` 或被拼错），导致请求挂起/失败。
4. **RLS/权限差异**：列表接口可读但详情接口的 select 字段或 join 导致触发权限限制，返回 401/403。
5. **前端运行时异常**：详情页渲染阶段抛错（例如对 `null` 解构），中断渲染但 loading 组件仍在。
